# Space-Invaders
This is a multiplayer version of SpaceInvaders in processing. This is my ICS3U1 summative. 

## Features
- scores are stored even after the game is over 
- 2 player
- powerups when you hit the spaceship
  - spread firingmode
  - 3 way firingmode
- invaders change sprites
- sound effects
- invaders fire back
